Run instructions (local)

Create folders and files as above. Suggested layout:

homecareai/
├─ backend/
│  ├─ app/
│  │  ├─ __init__.py
│  │  ├─ main.py
│  │  ├─ analysis.py
│  │  └─ models.py
│  └─ requirements.txt
├─ playground.html


Install and run:

cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000


Open playground.html in browser (file://) — it will call http://localhost:8000 endpoints (note: to avoid CORS issues opening file:// you can serve playground.html with a tiny static server: python -m http.server 5173 from repo root and open http://localhost:5173/playground.html).

Quick cURL tests

Upload video:

curl -F "file=@/path/to/your/fallen-women.mp4" http://localhost:8000/upload-video
# => returns {"video_id":"abc123","filename":"..."}


Analyze it:

curl -X POST http://localhost:8000/analyze/abc123
# => returns analysis JSON (events, rep frame path)


List events:

curl http://localhost:8000/events | jq


Fetch event:

curl http://localhost:8000/events/evt-001 | jq


Download representative frame:

curl http://localhost:8000/static/frames/abc123_rep.jpg -o rep.jpg
